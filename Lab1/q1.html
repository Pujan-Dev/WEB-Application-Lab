Paragraph with Text Formatting

This is a paragraph demonstrating the use of <b> bold </b> and <i> italic </i> text formatting.

You can also use <strong> strong </strong> for important text and <em> emphasis </em> for emphasized text. These are semantic tags that convey meaning to browsers and screen readers.

More Formatting Examples

The <b> tag makes text bold without semantic importance.
The <strong> tag makes text bold with semantic importance.
The <i> tag makes text italic without semantic meaning.
The <em> tag makes text italic with emphasis meaning.
